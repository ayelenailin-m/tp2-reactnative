## Trabajo Práctico - React 

Este proyecto consiste en hacer una página web con React con tema a elección. En este caso, elegí hacerlo con cosas sobre mí.

---

### Ejecución

1. Cloná el repositorio
2. Instalá dependencias
3. Ejecutá el proyecto

```bash
git clone https://github.com/ayelenailin-m/tp2-reactnative.git
cd tp2-reactnative
npm install
npm run dev
```
4. Abrí tu navegador e ingresar a http://localhost:5173/
